# My Todo App

Todo 관리를 위한 웹 애플리케이션입니다.

## Features

- Todo를 생성하고 삭제할 수 있습니다.
- Todo의 완료 상태를 변경할 수 있습니다.
- Github Project 칸반을 사용해본다.
- dev 브랜치를 만들어 main 브랜치로 Pull Request를 최소 1회 시도한다.
- Redux, Styled-Component, ESLint, Prettier를 설치해본다.
